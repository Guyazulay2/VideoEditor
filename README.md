# VideoEditor
**VideoEditor** is a video editing application that allows you to upload, cut, edit, and export video files easily through an interactive interface.

## Requirements
- Docker
- Docker Compose

## Installation and Run
Clone the project and run it using Docker Compose:

```bash
$ git clone https://github.com/Guyazulay2/VideoEditor.git
$ cd VideoEditor
$ docker-compose up --build
```

Then open the interface in your browser at http://localhost:3000. You can now start editing and exporting your video files easily.
